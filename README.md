# Product Metrics Dashboard (Streamlit)

A lightweight dashboard that generates *synthetic* DAU, retention, and conversion and visualizes trends to simulate PM decision-making.

## Why it's useful (APM angle)
- Communicates how you track core metrics and detect anomalies.
- Lets you experiment with growth/retention scenarios without real data.

## Setup & Run
```bash
pip install streamlit pandas numpy matplotlib
streamlit run app.py
```
